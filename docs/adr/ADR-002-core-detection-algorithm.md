# ADR-002: 使用注意力熵变率 + 隐藏态范数扫描作为核心检测算法

## Status

**Final**（实验已终止，结论见 docs/experiment-report.md）

## Context

本工具的核心问题是：如何量化 Prompt 文本的"结构健康度"？候选方案：

1. **注意力熵变率 + 隐藏态范数扫描**（最终选择）— 读取模型内部物理量
2. **Perplexity 评分** — 计算模型对文本的困惑度，越低表示越"自然"
3. **Saliency Map（梯度归因）** — 计算每个 Token 对输出的梯度贡献
4. **纯规则启发式** — 基于长度、重复率、标点密度等表面特征

选择方案 A+C 的理由：

- **Attention Entropy Delta**：浅层到深层的熵值突变标志着"句法依赖断裂"，有强物理意义。当深层比浅层熵突然飙升，说明模型在该处的注意力分散，需要在多个可能的句法结构中"硬猜"。
- **Hidden State Norm**：范数极低的 Token 在模型内部被"压缩坍塌"，无论 Attention 是否关注它，它都已失去语义激活强度。这对 RAG 长文本特别重要——拼接的段落可能在某些层被静默忽略。

Perplexity 作为整体指标无法定位到 Token 级问题；Saliency Map 需要梯度计算，计算量大且梯度噪声高；纯规则完全无法捕捉 Transformer 内部表征。

## Decision

同时实现两种互补的检测算法：

### 熵变率（方案 A）

**最终实现**（与原始设计有偏差，详见下方）：
- 计算**每个 Token 位置**的注意力熵差 `delta[i] = entropy(layer[-1][:,i,:]) - entropy(layer[-3][:,i,:])`
- 使用 z-score 异常检测 + 绝对阈值混合判定：
  - `abs(z) > 2.0` → 高风险（红色）
  - `abs(z) > 1.3` → 中风险（黄色）
  - `delta > 2.0/1.5` → 对应风险等级（绝对阈值兜底）
- 最终风险等级取 z-score 和绝对阈值中的较高者

**与原始设计的偏差**：
- 原定「只看最后一个 Token」→ 改为「逐位置分析」，否则中间位置的逻辑死结完全无法检测
- 原定「固定阈值 2.0/1.5」→ 改为「z-score 为主 + 绝对阈值为辅」，适应不同输入长度

### 范数扫描（方案 C）

**最终实现**：
- 提取 `hidden_states[-3]` 的 L2 范数
- 按 `chunk_size=32`（默认）聚合，计算每块的 `mean_norm` 和 `norm_std`
- **双重判定**：`mean_norm < 15%` 或 `norm_std < 15%` 任一满足即标记为弱信号

**与原始设计的偏差**：
- 原定「只看 mean_norm」→ 改为「mean + std 双阈值」，因为重复文本的均值差异小但方差剧烈坍塌
- 原定 `chunk_size=128` → 改为 `32`，128 下短文本只有 1 块无法比较

### 共享推理
两个算法共用同一个模型推理结果（一次 forward pass 同时输出 attention 和 hidden states），计算开销几乎不变。

## Consequences

- **更好**：从两个正交维度（句法 + 语义）覆盖文本问题，检测更全面
- **更好**：共享一次模型推理，性能开销小
- **更好**：逐位置熵差可以定位到具体的"逻辑死结"位置
- **更差**：算法物理意义需要向非技术用户解释，增加教育成本
- **更差**：所有阈值（z>2.0, percentile=15, chunk_size=32）均为启发式默认值，**无法泛化**到不同文本风格
- **更差**：实验证明 0.5B 模型无法区分"逻辑谬误"与"复杂指令"，信噪比不足。详见 `docs/experiment-report.md`

> **范数扫描算子约定（F1 修复）：** 弱信号判定采用**严格小于**（`norm_score < percentile_threshold`）。
> - `percentile=0` 时没有任何块满足 `norm < 最小值`，故 `weak_count == 0`
> - `percentile=100` 时阈值等于最大值，最大值块不满足严格 `<`，故 `weak_count == len - 1`
> 此约定已在 `test_zero_percentile` 和 `test_hundred_percentile` 中固化。

> **实验结论（追加）：** 该项目已终止。浅层物理量在工业级自然语言/代码混合输入下不可行。
> - 信噪比被任务复杂度污染，而非文本质量
> - 平均池化抹平了物理波动，chunk_size 调参才能观测
> - 失去 Linter"一键体检"的核心价值
